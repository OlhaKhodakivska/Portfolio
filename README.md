# 🚀 Creative Portfolio — Front-End Project

A responsive personal portfolio website built as a front-end project for **Digital Career Institute (DCI)**.

This project demonstrates practical frontend development skills using **semantic HTML**, **modern CSS**, and **Vanilla JavaScript**.  
The website includes a hero section, responsive navigation, portfolio showcase, services section, and interactive contact form.

```bash

```

---

You can run the project using:

- VS Code Live Server
- Any local development server
- Directly opening `index.html`
