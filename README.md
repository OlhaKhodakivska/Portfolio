## 📌 Project Overview

This is a responsive personal portfolio website built as a front-end homework assignment for **Digital Career Institute (DCI)**.

The project demonstrates practical skills in semantic HTML, modern CSS layout techniques, and vanilla JavaScript interactivity. It presents a hero section, services/capabilities, selected projects, and a functional contact form UI.

---

## 🛠 Tech Stack

- HTML5 (semantic markup)  
- CSS3 (Flexbox, variables, animations, responsive design)  
- Vanilla JavaScript (DOM, events, form handling)  
- GitHub Pages (deployment)

---

## ⚙️ Features

- Responsive navigation with hamburger menu  
- Hero section with adaptive Flexbox layout  
- Capabilities section using semantic `<article>` blocks  
- Portfolio cards with hover animations  
- Contact form with JavaScript handling  
- Dynamic current year in footer  
- Mobile adaptation via media queries
